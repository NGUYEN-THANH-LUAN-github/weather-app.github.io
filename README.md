# 🌱 weather app

a project from The Odin Project

**開発ツール・環境：**
- Openweather API Unsplash API
- Webpack
- SCSS

**プロジェクト説明：**
- 地域検索可
- 天気・温度、湿度、風速等展示
- １週間予報と２４時間予報展示
 
**応用した知識・技術：**
- JSON操作
- 非同期プログラミング
- Fetch API
- asyncとawait
